Test Task
=========

Prerequisites
-------------

* [Java Development Kit (JDK) 11](https://adoptopenjdk.net/)
* [Maven 3](https://maven.apache.org/download.cgi)

Build and Run
-------------

1. Run in the command line:
	```
	mvn spring-boot:run
	```

2. Open `http://localhost:8080` in a web browser.
